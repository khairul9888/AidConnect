AidConnect Web App
------------------

Tools: NetBeans 8.2, GlassFish 4.1.1, Java EE, Derby DB

1. Open project in NetBeans (File > Open Project)
2. Right-click > Run
3. If DB is missing:
   - COPY THE AidConnectDB into your >
   - C:\Users\<YourName>\AppData\Roaming\NetBeans\derby
4. Register a user and begin using the system.

Login URL: http://localhost:8080/AidConnect/login.jsp
